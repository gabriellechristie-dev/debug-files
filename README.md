
# Debug Files

Jogo investigativo em CLI onde cada caso representa uma fase.

O jogador analisa códigos bugados, investiga pistas e tenta descobrir a causa do problema para corrigir o sistema.

Cada caso foi pensado para treinar erros comuns e importantes em Python.

---

# Estrutura do Jogo

```text
MENU
 └── Selecionar caso
      └── Narrativa
           └── Código bugado
                └── Investigar pistas
                     └── Tomar decisão
                          └── Corrigir bug
                               └── Resultado
                                    └── Próximo caso
```
