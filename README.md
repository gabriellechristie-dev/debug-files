## Debug Files 
Jogo investigativo em CLI onde cada caso é uma fase.
O jogador analisa códigos bugados, investiga pistas e tenta encontrar a causa do problema para corrigir o sistema.
Cada caso treina um tipo importante de erro comum em Python.

## Estrutura do jogo
MENU
↓
Selecionar caso
↓
Narrativa
↓
Código bugado
↓
Investigar pistas
↓
Tomar decisão
↓
Corrigir bug
↓
Resultado
↓
Próximo caso

## Sistemas do jogo
vidas
energia
narrativa interativa
save em JSON
progressão por casos

## Casos planejados
Caso 01 — Sistema Bancário
Tema:
 Cálculo de saldo incorreto
Foco técnico:
variáveis
atribuição
operadores
cálculos errados

Caso 02 — Cadastro Infinito
Tema:
 Sistema travando usuários
Foco técnico:
loops
condição de parada
loop infinito

Caso 03 — Login Fantasma
Tema:
 Usuários válidos não conseguem entrar
Foco técnico:
condicionais
operadores lógicos
comparações incorretas

Caso 04 — Inventário Corrompido
Tema:
 Itens desaparecendo do sistema
Foco técnico:
listas
mutabilidade
remoção/alteração de dados

Caso 05 — O Caso Final
Tema:
 Sistema principal comprometido
Foco técnico:
mistura de bugs
debugging completo
análise lógica mais complexa
